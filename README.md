# kylemogged

Landing page for Kyle Mogged.

- Single static page (`index.html`) — no build step.
- Wordmark set in **Poiret One** (the Trailer Club display face).
- Palette: black / gold / purple.
- Contact: kyle@kylemogged.com

## Local preview

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy

Served via GitHub Pages from the `main` branch root.
